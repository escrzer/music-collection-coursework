# Музыкальная коллекция (курсовая работа по Python)

Учебный Python-пакет для управления музыкальной коллекцией.  
В рамках курсовой реализованы классы трека, исполнителя и плейлиста, а также консольная программа с меню.

## Структура проекта

- `music_collection/track.py` — класс `Track` (название, исполнитель, длительность).
- `music_collection/artist.py` — класс `Artist` (исполнитель и его треки).
- `music_collection/playlist.py` — класс `Playlist`:
  - добавление треков;
  - удаление треков;
  - поиск по исполнителю;
  - перемешивание плейлиста.
- `music_collection/cli.py` — консольное меню для работы с плейлистом.
- `bin/music-collection` — скрипт для запуска программы из командной строки.
- `setup.py` — файл настройки пакета.
